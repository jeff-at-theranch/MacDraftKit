# Contributing

MacDraftKit welcomes focused bug reports, sanitized sample documents, format observations, tests, and code contributions.

## Development requirements

- Keep `main` buildable.
- Run `swift test` before opening a pull request.
- Add a regression test before fixing a parser bug.
- Document public API.
- Separate observed facts about the MD70 format from hypotheses.
- Do not commit private or sensitive drawings.

## Pull requests

Prefer small pull requests with one purpose. Explain the observed file behavior, the implementation choice, and how the change was verified.
